# 🏥 AI-Powered Healthcare Prediction & Resource Management System

## 📌 Project Overview

The AI-Powered Healthcare Prediction & Resource Management System is an intelligent healthcare platform designed to assist hospitals in improving patient care, predicting disease risks, managing healthcare resources, and optimizing hospital operations.

The system uses Artificial Intelligence (AI), Machine Learning (ML), and Data Analytics to provide disease prediction, treatment recommendations, patient outcome forecasting, bed management, staff scheduling, and medical report analysis.

---

## 🎯 Objectives

- Predict disease risks using patient health data
- Recommend treatments based on predicted conditions
- Forecast patient recovery outcomes
- Manage hospital beds and resources efficiently
- Analyze medical reports automatically
- Provide AI-powered healthcare assistance
- Improve healthcare decision-making

# 📦 Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/AI_Healthcare_System.git
```

## Move to Project Folder

```bash
cd AI_Healthcare_System
```

## Install Dependencies

```bash
py -m pip install -r requirements.txt
```

---

# ▶️ Run Application

```bash
py -m streamlit run app.py
```

Application will run on:

```text
http://localhost:8501
```
